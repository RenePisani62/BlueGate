ADR-0005 – Application Lifetime Managed by BlueGateAgent

Decision

The application's execution lifecycle is managed by BlueGateAgent. Program.cs is responsible only for application startup and dependency creation.

Rationale

Separates orchestration from the entry point.
Enables reuse by a future Windows Service.
Simplifies testing by separating Run() from RunOnce().
Supports future hosting models (console, service, Linux daemon).
