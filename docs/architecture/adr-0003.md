ADR-0003

Adopt Rule-Based Detection Architecture

Context
BlueGate initially performed detections using static helper methods. As the number of detection rules grows, this approach becomes difficult to maintain.

Decision

Introduce the IDetectionRule interface. Detection rules are now independent components managed by the DetectionEngine.

Consequences

Easy to add new rules.
Detection Engine remains unchanged as functionality grows.
Rules become independently testable.
Supports future plugin architecture.
